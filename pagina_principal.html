Sobre Mí

Mi aventura en el desarrollo comenzó por curiosidad y se convirtió en una verdadera pasión. Empecé de forma autodidacta, explorando tutoriales y documentación, hasta que logré crear mi primer proyecto funcional. Ese momento de ver código transformarse en una aplicación real fue transformador para mí.

Aunque mi experiencia proviene principalmente de proyectos personales, cada uno me ha enseñado lecciones valiosas sobre arquitectura de software, experiencia de usuario y resolución de problemas. He aprendido a depurar código, optimizar rendimiento y, sobre todo, a perseverar cuando las cosas no salen como esperaba.

    habilidades

  • React & JavaScript Moderno: Desarrollo de interfaces interactivas y componentes reutilizables usando React, hooks y JavaScript ES6+. Manejo de estado, routing y integración con APIs.
  • Node.js & Express: Creación de servidores backend robustos, desarrollo de APIs RESTful, middleware personalizado y manejo de autenticación básica.
  • Base de Datos & Persistencia: Diseño e implementación de esquemas de datos usando PostgreSQL/MongoDB, operaciones CRUD y modelado de relaciones entre entidades.
